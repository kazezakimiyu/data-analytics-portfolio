## Data Cleaning Script

This script processes raw food inventory data into a structured format suitable for analysis.

### Key Features

• Standardizes inconsistent item descriptions  
• Applies rule-based category mapping  
• Cleans and formats text fields  
• Outputs structured data for SQL queries and dashboard use  

### Why It Matters

Raw data from multiple sources contained inconsistencies that made direct analysis unreliable.  
This script ensures data quality and consistency, enabling accurate aggregation and reporting.
