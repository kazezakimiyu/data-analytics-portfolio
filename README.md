# **Data Analytics Portfolio – Ruixin Li**

## Welcome to my data analytics portfolio. This repository contains selected projects demonstrating my experience with SQL, Python, and data visualization across real-world datasets.

My work focuses on:

- Data cleaning and transformation

- SQL-based data analysis

- Interactive dashboards for decision-making

- Automation of data workflows

## Projects
### Food Supply Gap Analysis
Analysis of statewide food distribution data to identify supply-demand imbalances and support resource allocation decisions for nonprofit food shelves.

***Key highlights***

- Built SQL queries in AWS Athena to analyze inventory and distribution metrics across 230+ food shelves
- Developed Python scripts to standardize inconsistent food shelf records using rule-based logic
- Reduced reporting preparation time from 2 days to 3 hours through automated data processing
- Designed Tableau dashboards to visualize regional food supply gaps and operational benchmarks

Tools: Python, SQL (AWS Athena), Tableau

### Washington State EV Market Analysis
Exploratory analysis of electric vehicle adoption trends using Washington State vehicle registration data.

***Key highlights***

- Analyzed 100,000+ EV registration records using SQL queries and hierarchical data modeling
- Investigated adoption patterns by vehicle make, model, and geographic region
- Performed time-series analysis to explore EV market growth by vehicle year
- Built interactive visualizations to examine regional adoption differences

Tools: SQL, Tableau

### Classic Models Sales & Profit Dashboard
Business intelligence dashboard analyzing revenue performance across product categories, countries, and time periods.

***Key highlights***

- Aggregated sales and profit metrics using MySQL queries across multiple relational tables
- Created Power BI dashboards with KPI tracking including MoM growth and YTD revenue
- Built decomposition trees for granular analysis of revenue drivers by country and product category
- Enabled drill-down exploration of product-level performance

Tools: MySQL, Power BI, DAX

### Pharmacological Data Pipeline
Python-based data pipeline designed to automate the collection and processing of pharmacological compound data.

***Key highlights***

- Developed web scraping scripts using Selenium and BeautifulSoup to extract compound data from PubChem
- Reduced manual data collection time from several days to 15 minutes
- Standardized chemical nomenclature and InChI strings for 1,000+ compounds
• Implemented error-handling logic to manage dynamic web elements during data extraction

Note: Due to research confidentiality, this repository contains a simplified version of the pipeline using publicly available data.

Tools: Python, Selenium, BeautifulSoup
